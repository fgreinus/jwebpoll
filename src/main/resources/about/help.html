Umfragen anlegen und entfernen


Auf den Buttons im linken Teil des Fensters ganz unten können Umfragen hinzugefügt bzw. entfernt werden.

Fragen hinzufügen


Ist eine Umfrage ausgewählt können über den Button "Frage hinzufügen" Fragen hinzugefügt werden.
Frage
Bestimmt die eigentliche Frage. Beispiel: Wenn Sonntag Bundestagswahl wäre, wen würden sie wählen?
Hinweis
Kann die gestellte Frage präzisieren. Beispiel:
Pflichtfrage
Gibt an ob die Frage beantwortet sein muss um den Fragebogen abgeben zu können.
Fragentyp
Es gibt 3 Frage Typen:
Singlechoice
Es kann aus den gegebenen Antwortmöglichkeiten nur eine ausgewählt werden.
Multiplechoice
Es können mehrere Antwortmöglichkeiten angekreuzt werden
Freitext
Es kann ein beliebiger Text angegeben werden
Antwortmöglichkeiten
Die Antwort Möglichkeiten zu einer Frage. Beispiel: Wenn Sonntag Wahl wäre, wenn würden sie wählen? A: SPD B: CDU C:Piratenpartei
Wert
Gibt die Gewichtung einer Frage an. Beispiel: Es gibt einen Fragebogen für die Notenvergabe. Die Codequalität soll doppelt so stark gewichtet werden wie das Design.
Webadresse
Dia Adresse unter der andere Computer im Netzwerk die Umfragen aufrufen können.
Es kann die Webadresse ausgewählt werden unter der die Umfrage erreichbar sein soll.
Hilfreich ist das beispielsweise wenn mehrere Netzwerk Interfaces aktiv sind oder man IPv6 verwenden will.

Freigeben und Schließen von Umfragen


Der Status einer Umfrage (freigegeben oder geschlossen) gibt an ob diese gerade auf der Website zur Beantwortung bereit steht.
Es kann immer nur eine Umfrage gleichzeitig geöffnet werden.

Ergebnisse und Auswertung


Der Button Ergebnisse öffnet die Auswertung. Es wird angezeigt, welche Antwortmöglichkeit wie oft gegeben wurde. Außerdem werden alle Antworten der Freitext Fragen gezeigt.
